import Paragraph from '@components/content/Paragraph.jsx';
import Link from '@components/content/Link.jsx';

<Paragraph>
  Meu portifólio é uma maneira de me apresentar e, ao mesmo tempo, mostrar
  minhas habilidades, todos os meus projetos de código aberto vão ser hospedados
  aqui. A identidade visual foi inspirada no site principal da Vercel. E para
  facilitar todo o processo de desenvolvimento utilizei o framework{' '}
  <Link href="https://astro.build/">Astro</Link> que me permitiu escolher qual
  biblioteca eu queria para a interface, a escolha foi o React.
</Paragraph>

<Paragraph>
  O motivo dessa escolha foi a minha experiência prévia com a biblioteca fazendo
  experimentos e pequenos projetos utilizando{' '}
  <Link href="https://nextjs.org/">NextJS</Link>. Isso foi importante, pois eu
  consegui usar todas as ferramentas nas quais tenho estudado e mostrar minhas
  habilidades com elas. Acho que a parte mais demorada desse projeto foi a
  idealização dele, me levou algumas boas semanas no Figma, mas no final valeram
  os esforços.
</Paragraph>

<Paragraph>
  A biblioteca CSS Tailwind foi escolhida para a estilização por ser rápida,
  simples e escalável. É muito mais fácil focar no desenvolvimento de novas
  features e melhorias quando não preciso ficar pensando onde vou colocar cada
  arquivo CSS e como vou organizá-los. Um arquivo único é gerado na build com
  todas as otimizações necessárias, sem dor de cabeça (em boa parte do tempo).
</Paragraph>

<Paragraph>
  Tudo isso levou entre 4 e 5 semanas para ficar pronto sem contar todo o
  planejamento do Figma, todo o código está disponível sobre a licença MIT e
  você pode dar uma olhada{' '}
  <Link href="https://github.com/franciscocandido/flyrio.dev">aqui</Link>.
</Paragraph>
